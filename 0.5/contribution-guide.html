# How to contribute?

Follow these steps:

  • Go to: https://github.com/runeasymail/easymail
  • Fork the master branch.
  • Create your feature branch (git checkout -b my-new-feature).
  • Commit your changes (git commit -am 'Added some feature').
  • Push to the branch (git push origin my-new-feature).
  • Create a new Pull Request.

# Which Branch?

The pull requests must be sent to the master branch.

# Other ways to contribute

Every individual, company or organization who uses and likes EasyMail can support it by providing their names, logos and other initials in order to be used on the website, the Github repository and advertisement materials whose purpose is to popularize EasyMail.

You can also donate to the project via PayPal using the button below. The money collected through the donations will help us to keep up the project and to pay things like domain name, development tools used during the development and also for the popularization of the project and the growth of the community.
